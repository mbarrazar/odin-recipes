# odin-recipes
Este proyecto consiste en la creación de un sitio web básico de recetas utilizando únicamente HTML.

Al completar este proyecto, habré demostrado habilidades fundamentales como:

- Crear la estructura básica de páginas HTML.

- Trabajar con enlaces relativos y absolutos.

- Insertar imágenes en una página web.

- Organizar archivos y carpetas de un proyecto.

- Utilizar Git y GitHub para el control de versiones.

El objetivo principal es practicar y reforzar los conceptos básicos de HTML aprendidos hasta ahora en The Odin Project.
